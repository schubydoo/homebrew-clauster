# homebrew-clauster

Homebrew tap for [clauster](https://github.com/schubydoo/clauster) — a self-hosted
web UI for spawning and managing Claude Code remote-control bridges.

```sh
brew install schubydoo/clauster/clauster
```

The formula installs the signed standalone binary from clauster's GitHub releases.
It is regenerated automatically on each release by clauster's `packaging-bump`
workflow; manual edits will be overwritten.
